## Create new contract
```shell
truffle create contract {CONTRACT_NAME}
```

## Create migration
```shell
truffle create migration {MIGRATION_NAME}
```

## Deploy
```shell
truffle migrate --network {CHAIN}
```

## Verify contract on BSC Scan
```shell
truffle run verify {CONTRACT_NAME}@{CONTRACT_ADDRESS} --network {CHAIN}
```
